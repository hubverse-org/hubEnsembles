# Example weights data for `linear_pool()`

Toy weights data formatted according to hubverse standards to be used in
the examples for
[`linear_pool()`](https://hubverse-org.github.io/hubEnsembles/dev/reference/linear_pool.md).
Weights are 0.25, 0.5, 0.25.

## Usage

``` r
weights
```

## Format

### `weights`

A data frame with 3 rows and 2 columns:

- model_id:

  model ID

- location:

  FIPS codes

- weight:

  weight
