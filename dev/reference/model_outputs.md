# Example model output data for `simple_ensemble()`

Toy model output data formatted according to hubverse standards to be
used in the examples for
[`simple_ensemble()`](https://hubverse-org.github.io/hubEnsembles/dev/reference/simple_ensemble.md)

## Usage

``` r
model_outputs
```

## Format

### `model_outputs`

A data frame with 24 rows and 8 columns:

- model_id:

  model ID

- location:

  FIPS codes

- horizon:

  forecast horizon

- target:

  forecast target

- target_date:

  date that the forecast is for

- output_type:

  type of forecast

- output_type_id:

  output type ID

- value:

  forecast value
